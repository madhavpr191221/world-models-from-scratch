Overview

What this page is for

This project looks at a self-supervised vision model. The site shows what the model is, how it is checked, and how it behaves on image retrieval.

  1. Train a vision encoder with unlabeled images.
  2. Freeze the encoder and probe the features.
  3. Use retrieval to see which images end up near each other.
