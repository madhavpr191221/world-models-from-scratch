What to look at

The main point is whether the learned features can be read out and whether nearby images look related.

  1. Linear probe: checks if class labels are readable from features.
  2. Low-shot probe: checks what happens with less labeled data.
  3. k-NN: checks if nearby embeddings often share a label.
  4. Projector comparison: checks whether the projector behaves the same way as the encoder.
